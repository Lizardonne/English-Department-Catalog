English Department Catalog

Coded, compiled, and organized by Alex Slansky


This project overlaps three connected interests:

  • CS260's creative projects,
  • The TA HumGrant, and
  • ENGL394R's dependence on that project.

This site is where the project will be hosted--at least for the Fall 2020 semester.

Loosely, the steps of the project are:

  1. Create a mock-up of the database; define the information that will be stored, and the tables in which it will be stored.
  2. Gather information; import catalog and directory information to the database structure, then email professors for missing information and confirmation. (Would a Google Survey be helpful here?)
  3. Write the database; cf. CS240's FamilyMap database assignment.
  4. Compile gathered information into the database.
  5. Integrate the database with the website; may need to wait for instruction on MongoDB in CS260.
